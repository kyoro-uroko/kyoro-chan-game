# キョロちゃんの冒険 🐦

ウロコインコの「キョロちゃん」を操作して、後ろから追ってくるタカから逃げきる
エンドレス・スコアアタックゲームです。ブラウザだけで動く1ファイル（`index.html`）。

## 遊び方
- **タップ / クリック / スペース**ではばたく（フラッピー操作）
- 🌾 餌を食べると **SPEED が上がってタカを引き離せる**（取り逃すとタカが食べて加速）
- ✨ ひとかたまりの餌を**全部食べると NICE！**、7回で **TURBO（無敵＋最高速）**
- 🌟 流れ星をキャッチすると**即TURBO**
- 障害物（木・家・ビル・電柱・ブイ・岩・看板・雲）と 🦅タカ・🐦カラス・🚗車・⛵船 を避ける
- 5000mごとに **街 → 森 → 海** とエリアが変化。空は60秒周期で昼夜が移ろう

逃げた距離（m）がスコア。ハイスコアは端末に保存されます。

## 起動
`index.html` をブラウザで開くだけ（スマホでもOK）。
